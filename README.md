# Liquid Glass Components

Beautiful, interactive WebGL-powered components with stunning glassmorphism effects.

Inspired by Apple's iOS 26 Liquid Glass design.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![WebGL](https://img.shields.io/badge/WebGL-enabled-green.svg)

## Features

- Ultra-realistic glassmorphism with WebGL
- Highly customizable effects
- Smooth GSAP animations
- Responsive design
- Hardware-accelerated rendering

## Components

### Liquid Glass Header

Interactive header with WebGL-powered glassmorphism effects.

[View Demo](./liquid-glass-header/demo.html) | [Documentation](./liquid-glass-header/README.md)

## Browser Support

Currently supports **Chrome (latest version)** only. Firefox, Safari, and Edge support coming soon.
